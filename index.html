Resnet-18/50

I am interested in machine learning and artificial intelligence so much so that I chose it as a course. This project compares and evaluates convolutional neural networks (CNN) that uses a Resnet-18 trained on a 6000 image dataset with 3 categories of cancer cells and another pre-trained Resnet-50. This project was done using PyTorch, Scikit-Learn, Numpy, MatplotLib and Pandas.


Models and Datasets

The Resnet-18 was trained on dataset 1 while the Resnet-50 is the available Resnet-50 from PyTorch. Both models are then tested on datsets 2 (other type of cancer cells with 3 categories) and dataset 3 (unapologetically cute cats, dogs and wildlife) for feature extraction using t-SNE.


The Resnet-18 was trained on dataset 1 while the Resnet-50 is the available Resnet-50 from PyTorch. Both models are then tested on datsets 2 (other type of cancer cells with 3 categories) and dataset 3 (unapologetically cute cats, dogs and wildlife) for feature extraction using t-SNE.

Evaluating Resnet-18

Furthermore, the Resnet-18's feature extraction of Dataset 2 is evaluated using supervised learning and unsupervised learning, Support Vector Machine (SVM) and a K-Nearest Neighbors (KNN) classifier.
